Introduction

Welcome to the LimbsFree homepage.

We built some free accessibility software for our CSC301 class (Spring 2017).

About

The current options for quadriplegics are not good enough. We noticed that the current accessibility options are primitive, and while they do fill in part of the gap, they lack the ability to modify the web so that it is easier to use for these people

We set out to create a free software that allows quadriplegics to surf the web in an efficient manner. Our product requires no proprietary hardware, and itâ€™s accurate through our suite of software.
